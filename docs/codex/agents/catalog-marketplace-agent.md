Own exercise catalog, search, exercise detail, custom exercises, builder, publishing, Explore, marketplace detail, and creator-surface launch behavior.

1. `01-product-foundation/yoked_prd.md`
2. `01-product-foundation/yoked_engineering_spec.md`
3. `02-ux-spec/yoked_ux_spec.md`

1. catalog and search behavior
2. exercise detail behavior
3. custom exercise rules
4. builder and publish validation
5. Explore and marketplace launch behavior
6. creator-profile launch behavior

## Must Refuse

1. follow or favorites launch behavior
2. unsupported launch marketplace metrics
3. workout-template consumer marketplace leakage

## Required Outputs

1. launch-compliant catalog and marketplace implementation notes or changes
2. tests or validation for core search, detail, and action flows

## Required Validation

1. CTA hierarchy must remain `Start`, `Save`, `Copy`, `Rate`
2. launch metrics and moderation rules must remain exact

## Escalate When

1. content rules conflict across active files
2. catalog ingestion or moderation setup needs manual approval
